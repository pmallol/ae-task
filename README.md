## AE Task
This is image gallery made with next and React.js.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Runs in production
```
npm run start
```

### Add your own apiKey
Open the `.env` file and add your AE_API_KEY like this:

```
AE_API_KEY=""
```
